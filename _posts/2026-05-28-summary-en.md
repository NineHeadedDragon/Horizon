## [Go Proposes Generic Methods, Approved for Implementation](https://github.com/golang/go/issues/77273) ⭐️ 8.0/10

A proposal by Go co-designer Robert Griesemer to add generic methods to Go has been approved and is moving to implementation, reversing a longstanding position in the language's FAQ. This addresses a major limitation in Go's generics, enabling developers to write more reusable and expressive code, especially for libraries and frameworks. It brings Go closer to modern generics support found in other languages like Java and C#. The proposal allows methods to have their own type parameters, separate from the type's parameters. However, Go interfaces still cannot include generic methods, which remains a separate constraint.

hackernews · f311a · May 27, 09:02 · [Discussion](https://news.ycombinator.com/item?id=48291575)

**Background**: Go added generics in version 1.18, allowing functions and types to have type parameters, but method-level type parameters were explicitly excluded. This limitation was often cited as a significant gap, leading to awkward workarounds. The reversal shows the language's continued evolution toward more flexibility while maintaining simplicity.

The project has long required legal paperwork to place contributions in the public domain.

## [Microsoft Copilot Cowork Email Exfiltration Attack](https://simonwillison.net/2026/May/26/copilot-cowork-exfiltrates-files/#atom-everything) ⭐️ 8.0/10

A security vulnerability in Microsoft Copilot Cowork allows agents to exfiltrate files by sending emails containing external images that leak data when opened. Prompt injection can exploit this to leak pre-authenticated OneDrive download links. This vulnerability highlights a critical challenge in securing agentic AI systems, where automated actions can be exploited for data exfiltration. It affects Microsoft Copilot Cowork users and underscores the need for better access controls in AI products. The attack works because Copilot Cowork agents can send emails directly to the user's inbox without approval, and those emails render external images that trigger network requests. Additionally, OneDrive's pre-authenticated download links allow attackers to download files if leaked via prompt injection.

rss · Simon Willison · May 26, 15:36

**Background**: Agentic AI systems are designed to perform actions on behalf of users, such as sending emails or accessing files. However, they are vulnerable to prompt injection attacks, where malicious instructions cause the AI to perform unintended actions. Data exfiltration via external image rendering is a known technique in AI products, as seen in previous vulnerabilities in Continue and Cursor IDE.

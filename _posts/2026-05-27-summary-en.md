This vulnerability underscores the persistent challenge of securing agentic AI systems against prompt injection, potentially affecting millions of Microsoft 365 users and eroding trust in AI-powered productivity tools. The attack exploits the fact that Copilot Cowork can send emails to the user's own inbox without approval, and those emails can contain externally hosted images that trigger network requests, leaking data. OneDrive's pre-authenticated download links further enable file theft.

rss · Simon Willison · May 26, 15:36

**Background**: Prompt injection is a cybersecurity exploit where malicious inputs cause large language models (LLMs) to disregard safeguards and execute unintended actions. In agentic systems like Copilot Cowork, which can access files and send emails, indirect prompt injection via web content is a particular risk. Rendering external images in emails is a known data exfiltration technique, as the image request can include sensitive data as URL parameters.

## [7-Zip Heap Overflow Vulnerability Allows Code Execution](https://socprime.com/blog/cve-2026-48095-7-zip-heap-overflow-flaw/) ⭐️ 8.0/10

A heap overflow vulnerability (CVE-2026-48095) in 7-Zip's NTFS handler has been publicly disclosed, allowing arbitrary code execution; the issue was fixed in version 26.01 released on April 27, 2026. Given 7-Zip's widespread use for file compression, this high-severity vulnerability (CVSS 8.8) poses a significant risk, especially because crafted files with misleading extensions (e.g., .7z, .zip, .rar, or even no extension) can trigger the vulnerable NTFS handler via fallback logic, expanding the attack surface for phishing and social engineering. The vulnerability, discovered by Jaroslav Lobačevski of GitHub Security Lab, involves a heap buffer write overflow in the NTFS archive handler (NtfsHandler.cpp). The signature-based fallback logic can route files with arbitrary extensions to the vulnerable handler after the extension-matched handler fails, requiring no user interaction beyond opening the file.

telegram · zaihuapd · May 27, 08:01

**Background**: A heap overflow occurs when data is written beyond the allocated memory buffer, potentially corrupting adjacent memory and allowing an attacker to execute arbitrary code or crash the application. NTFS (New Technology File System) is Microsoft's default file system; 7-Zip includes an NTFS handler to extract files from NTFS disk images. The vulnerability exploits the handler's signature-based fallback: when 7-Zip encounters a file whose primary handler fails, it checks other handlers by file signatures, allowing a crafted NTFS image with any extension to be processed.
